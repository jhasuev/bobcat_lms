## Bobcat lms is an open source project. Web application to create online courses.

### Objectives: to train team software development 

### Technology stack:
 - Django/DRF (Backend)
 - VueJS (Frontend)
 - Docker/docker-compose
 
 

Build

    docker-compose build
Run

    docker-compose up
